# SafeZone AI 🛡️
Open Source Industrial Safety Analytics

## Overview
SafeZone AI is an open-source video analytics platform designed to improve workplace safety in industrial environments. Using computer vision, it monitors CCTV feeds to detect unauthorized entry into hazardous zones and provides real-time alerts.

## Problem
Industrial accidents often happen when workers enter restricted dangerous areas near heavy machinery, cranes, or chemical zones. Existing proprietary systems are expensive and often inaccessible for SMEs.

## Solution
SafeZone AI provides:
- Affordable open-source safety monitoring
- Compatibility with standard CCTV cameras
- Real-time hazard zone intrusion detection
- Privacy-focused local processing

## Features
- Hazard zone boundary monitoring
- Real-time alerts
- Configurable danger zones
- MIT open-source license
- Lightweight deployment

## Technology Stack
- Python
- OpenCV
- NumPy

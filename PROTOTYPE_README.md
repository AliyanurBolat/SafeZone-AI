# 📸 SafeZone AI - Visual Prototype

This section contains the visual proof-of-concept (PoC) for the **SafeZone AI** project. 

## 🎯 Purpose
The prototype demonstrates the core logic of our system: using computer vision to monitor industrial camera feeds, delineate safety zones, and detect personnel in restricted areas to prevent workplace accidents.

## 📂 Included Files
*   `prototype.py` - The Python script that uses the OpenCV library to generate the bounding boxes.
*   `test_image.jpg` - The original royalty-free image of the warehouse environment.
*   `Снимок экрана 2026-05-01 204332` - The final result demonstrating the active monitoring interface.
